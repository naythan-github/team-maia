# Tomorrow Planning Email

## Purpose
Morning email with strategic priorities, weather, calendar overview, and intelligent recommendations.

## Command Usage
```bash
# Generate for tomorrow
maia tomorrow_planning_email

# Generate for specific date
maia tomorrow_planning_email --date="2025-01-08"
```

## Agent Chain
1. **Calendar Analyzer**: Extract meetings and time blocks
2. **Priority Generator**: Analyze TODO items and current projects
3. **Weather Service**: Perth forecast integration
4. **Opportunity Scanner**: Check for new developments
5. **Email Composer**: Format and send morning brief
