# Analyze Prompt Command

## Purpose
Comprehensive analysis of prompt effectiveness with specific improvement recommendations.

## Usage
```
analyze_prompt <prompt_text>
```

## Process

### 1. Structure Analysis
- **Clarity Score** (1-10): How clear is the intent?
- **Specificity Level**: Are requirements well-defined?
- **Context Completeness**: Is sufficient background provided?
- **Output Specification**: Are desired outputs clearly defined?

### 2. Technical Evaluation
- **Token Efficiency**: Is the prompt concise without losing meaning?
- **Ambiguity Detection**: Identify unclear or multi-interpretable elements
- **Missing Elements**: What information would improve results?
- **Structural Issues**: Problems with organization or flow

### 3. Improvement Opportunities
- **High Impact Changes**: Most important improvements
- **Quick Wins**: Easy fixes with immediate benefit  
- **Advanced Optimizations**: Sophisticated enhancements
- **Template Potential**: Could this become a reusable pattern?

## Implementation Notes
- Use objective evaluation criteria
- Provide specific, actionable feedback
- Include both quick fixes and advanced optimizations
- Always deliver a ready-to-use improved version
