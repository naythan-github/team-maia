# Daily Achievement Email

## Purpose
Automated end-of-day email summarizing accomplishments, progress, and key outcomes.

## Command Usage
```bash
# Manual trigger
maia daily_achievement_email

# With custom date
maia daily_achievement_email --date="2025-01-06"
```

## Implementation Approach
1. **Data Collection**: Scan git commits, email activity, calendar events
2. **Content Generation**: Template-based with intelligent extraction
3. **Email Delivery**: Via existing Gmail integration
4. **Scheduling**: Cron job at 6:30 PM daily

## Agent Chain
1. **Data Collector**: Gather activity from multiple sources
2. **Content Generator**: Apply template with achievements
3. **Email Sender**: Format and send via Gmail
