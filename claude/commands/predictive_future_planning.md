## Command Parameters

### Prediction Parameters
- `--career-trajectory`: Predict career advancement and opportunities
- `--market-forecast`: Analyze market trends and opportunities
- `--resource-optimization`: Optimize resource allocation and planning
- `--comprehensive-forecast`: Generate integrated multi-domain forecast

### Timeline Parameters
- `--horizon=<months>`: Prediction horizon (1-24 months)
- `--timeline=<periods>`: Specific timeline analysis (immediate, 3months, 6months, 12months)
- `--target-date=<date>`: Specific target date for predictions

### Focus Parameters
- `--target-role=<role>`: Focus on specific career role
- `--industry=<industry>`: Industry-specific analysis
- `--location=<location>`: Geographic market analysis
- `--domains=<list>`: Specific domains to analyze

### Analysis Parameters
- `--confidence-threshold=<percentage>`: Minimum confidence level for recommendations
- `--scenario-analysis`: Include multiple scenario projections
- `--risk-assessment`: Include comprehensive risk analysis
- `--recommendations`: Generate actionable recommendations
